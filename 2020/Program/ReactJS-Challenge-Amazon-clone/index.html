Header 實作導覽頁

有查過的ＣＳＳ

position: sticky 是為了 User 在 Scroll Screen 的過程中固定 Header 在整個畫面中的位置

參考資料：https://ithelp.ithome.com.tw/articles/10194075

object-fit: contain 其作用為：在不影響外 container 情況下根據ＣＳＳ屬性要求進行填充
參考資料：
https://codertw.com/程式語言/740631/ > https://ddstudio.tw/css-object-fit/

關於 !important 的使用則是忽略ＣＳＳ渲染優先層級，直接指定該屬性設定為最優先

參考資料：

https://medium.com/ui-ux練功坊/css的優先順位-cfb859e988bd

Home 頁面主體內容

mask-image 圖片遮罩效果，大抵概念建立在圖像漸層上，藉由 masks 屬性以達到透視、遮罩、滑動動畫等等一系列特效效果，相類似效果還可以透過 background 屬性來完成

這段落組件ＣＳＳ使用都還算一般，比較特別的是其中 rating 的部分使用了一個還蠻有趣的 Array function
