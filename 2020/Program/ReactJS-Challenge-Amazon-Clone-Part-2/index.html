React Router DOM

一款由 React Router 延伸出來的套件，基本功能是 Router, Switch, Route, Link，簡單看了一下 Google 的反饋的資料，大致概念應該是透過 Router、Switch、Route 分配頁面渲染的區塊優先順序，已達到資源效率化

Router 似乎還有整合 ＡＰＩ串接上的功用，更詳細的內容得花點時間研究
Link 作用很像 a tag 裡面的 href ，但語法上簡潔許多

Left Part

.checkout 內使用了一個沒過的ＣＳＳ屬性max-content 簡單 google 了一下，是 CSS3 中出的四個屬性之一

CSS3 的四個屬性

fill-availabel 主要作用是填滿 div 裡面的空間，使用上就跟 width:100% 差不多意思，也可以作用在 height

fit-content 將元素寬度收縮成內容寬度，我的理解應該是類 padding 的空間收縮到內容物等寬的程度

max-content采用內部元素寬度值最大的那個元素的寬度作為最終容器的寬度，我是把這段話當成類似無條件進位的概念，取該元素內最大寬度內容為最終值

min-content 采用內部元素最小寬度值最大的元素單位寬度，例如 img 中的圖片渲染時的初始寬度，p 、h1這類文本元素內的文字寬度，中文就是以一個字為基本單位，所以最大寬度理解相對簡單，英文則是默認一個單字為單位，在該 tag 中的文本會以最長的單字為取樣單位
